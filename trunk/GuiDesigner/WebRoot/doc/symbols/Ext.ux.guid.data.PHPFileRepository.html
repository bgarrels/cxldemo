Class Ext.ux.guid.data.PHPFileRepository

Package:Ext.ux.guid.data
Defined In:Ext.ux.guid.data.PHPFileRepository.js
Class:PHPFileRepository
Extends:Ext.ux.guid.data.Repository
PHPFileRepository is a repository using phpFiles.php as a server side file repository. Files relative to rootBase can be loaded, changed or deleted by this repository.
