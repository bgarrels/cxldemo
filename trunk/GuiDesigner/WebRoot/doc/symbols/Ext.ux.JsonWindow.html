Class Ext.ux.JsonWindow

Package:Ext.ux
Defined In:Ext.ux.JsonWindow.js
Class:JsonWindow

JsonWindow implements a way to load a json directly into a window The window config elements of this window for example x,y can be set by specifing the values in the json tag
